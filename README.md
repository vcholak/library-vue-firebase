# library-vue3

## Project setup
```
pnpm install
```

### Start the development server
```
pnpm dev
```

### Compiles and minifies for production
```
pnpm build
```

### Preview the production build locally
```
pnpm serve
```

### Run your unit tests
```
pnpm test:unit
```

### Run your end-to-end tests
```
pnpm test:e2e
```

### Lints and fixes files
```
pnpm lint
```

See [Configuration Reference](https://vitejs.dev/config/).
